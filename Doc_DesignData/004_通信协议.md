# 协议简介

1. 双帧头发送
    - (FrameHeader) 第1个字节固定为0x53
    - (FrameType) 第2个字节则根据请求帧、应答帧、事件帧的不同而不同
        - (RequestFrame) 请求帧：0xCA
        - (ResponseFrame) 应答帧：0x35
        - (EventFrame) 事件帧：0x96
2. (PayloadSize) 然后就是4个字节，表示数据长度，为大端字节序
3. (PayloadData) 然后就是主要数据部分，其长度为上一步中指定的长度
    - (CommandType) 数据部分的前2个字节用来表示指令类型
4. (CRC16) 最后就是2个字节的CRC16校验，也为大端字节序
